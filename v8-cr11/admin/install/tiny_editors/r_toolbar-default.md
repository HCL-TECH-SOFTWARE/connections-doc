# The default toolbar configuration {#toolbar-default .reference}

The default toolbar used by the Tiny editors.

```
  toolbar: [
      'undo',
      {
        label: 'group.insert',
        items: [
          {
            id: 'insert',
            label: 'menu.insert',
            items: [
              [
                'link',
                'conn-insert',
                'bookmark',
                'media',
                'table'
              ],
              [
                'specialchar',
                'hr'
              ]
            ]
          }
        ]
      },
      'style',
      'emphasis',
      'align',
      'listindent',
      'format',
      [
        'conn-other',
        'conn-emoticons',
        'conn-macros'
      ],
      'language',
      'tools'
    ],
```
