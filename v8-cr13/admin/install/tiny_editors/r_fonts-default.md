# Default fonts list {#fonts-default .reference}

The default list of fonts used by the Tiny editors' font picker.

```sh
  fonts: [
    'Andale Mono',
    'Arial',
    'Arial Black',
    'Book Antiqua',
    'Comic Sans MS',
    'Courier New',
    'Georgia',
    'Helvetica',
    'Impact',
    'Symbol',
    'Tahoma',
    'Terminal',
    'Times New Roman',
    'Trebuchet',
    'Verdana',
    'Webdings',
    'Wingdings'
  ],
```
